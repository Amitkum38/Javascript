Difference between "var" and "let" Keyword ?

  1. "let" was introduced in ES2015/ES6
  2. "var" is there in JS from the beginning
  3. "let" has Block scope
  4. "var" has function scope
  5. "var" gets hoisted -
  6. Example :- let x = function(){ if(true){ console.log(a); console.log(b); var a = 10; let b = 20; } console.log(a); console.log(b) } x();
